# 📊 NAVEGANTES AI

> O painel de controle analítico e motor de IA do ecossistema Navegantes.

Este repositório contém o código-fonte do **NAVEGANTES AI**, focado no processamento de dados e visualização através de um dashboard interativo. 

🔗 **Acesse o Dashboard rodando ao vivo:** [Hugging Face Space](https://pmnvt-dashv3.hf.space)
🌐 **Voltar para o Portal Principal:** [Portal Navegantes](https://mlmpbr.github.io/navegaai_sobre/)

## 🎯 Objetivo do Projeto
Fornecer um ambiente robusto e eficiente para análise de dados e interação com nossos modelos, servindo como a principal ferramenta analítica do nosso ecossistema.

## 🛠️ Tecnologias Utilizadas
* Python
* Pandas, Numpy, Plotly, Dash
* Hugging Face Spaces (Hospedagem)
